### Getting Started

1. [Install nvm](https://github.com/creationix/nvm)
1. Run ```nvm install``` to set the correct Node version
1. Run ```npm install``` to install dependencies.
1. [Globally install Gulp](https://github.com/gulpjs/gulp/blob/master/docs/getting-started.md)
1. Run ```gulp``` to create minified file (outputs into ```./dist``` folder).
